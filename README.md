## Cuatro Amigos

Este es el repositorio de Cuatro Amigos.

Este proyecto presenta un juego de cartas española que destaca por su dinamismo, y se pone a prueba tu habilidad,
estrategia y la suerte donde haces lo posible para ganar, e intentas impedir que tus rivales ganen. El objetivo es ofrecer una experiencia divertida, competitiva y 
lo suficientemente impredecible para que cada ronda sea una experiencia única. Este juego viene con la característica de ser un crossover entre varios personajes de la ficción.

## Descripción
Cuatro Amigos es un juego de cartas de cuatro jugadores basado en estrategia, suerte y sabotaje. Diseñado para partidas rápidas, iteracción constante entre jugadores, y más.

## Términos y condiciones

Si usas, modificas o distribuyes este proyecto:

- Da crédito al repositorio original y a sus desarrolladores.
- Puede ser utilizado para fines educativos o recreativos.

## Guía del repositorio

El proyecto está organizado de la siguiente manera:

- **assets/**    Recursos gráficos
- **bin/**       Ejecutables o librerías
- **docs/**      Documentación del juego
- **libs/SFML**  Carpeta de librerías SFML
- **src/**       Código principal del juego
